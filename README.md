### Installation

```bash
brew install romnn/tap/cargo-publish-crates

# or install from source
cargo install cargo-publish-crates
```

### Development

Run the cargo plugin locally:

```bash
cargo run -p cargo-publish-crates -- --path ./path/to/crate
```
